Что такое виртуальная машина? -  программная система,исполняющая некоторый машинно-независимый код (например, байт-код) для реального процессора.

К какому типа языка программирование относиться Java? Компилируемый, объектно-ориентированный язык програмирования со статической типизацией переменных.

Из каких компонентов состоит Java? Язык програмирования Java, Платформа Java JRE - состоит из Java Virtual Machine (JVM), базовых классов платформы Java и вспомогательных библиотек платформы Java

Для чего используется JDK? комплект разработчика приложений на языке Java, включающий в себя компилятор Java (javac), стандартные библиотеки классов Java, примеры, документацию, различные утилиты и исполнительную систему Java (JRE).

Для чего используется JRE? необходимая для исполнения Java-приложений среда, без компилятора и других средств разработки. Состоит из виртуальной машины — Java Virtual Machine — и библиотеки Java-классов.

Для чего используется VM? Виртуальная машина Java исполняет байт-код Java, предварительно созданный из исходного текста Java-программы компилятором Java (javac)

Расскажите про базовые типы.  Типы в Java распределены на две категории: простые (примитивные) и [ссылочные (объектные)](Создавая объект его необходимо связывать с подобной ссылочной переменной). 
Что такое примитивные типы? В Java существует 8 примитивных типов данных:

byte (целые числа, 1 байт)
short (целые числа, 2 байта)
int (целые числа, 4 байта)
long (целые числа, 8 байтов)
float (вещественные числа, 4 байта)
double (вещественные числа, 8 байтов)
char (символ Unicode, 2 байта)
boolean (значение истина/ложь, 1 байт)
Эти 8 типов служат основой для всех остальных типов данных. Примитивные типы обладают явным диапазоном допустимых значений. 
эти типы доступны в рамках самого языка, для их использования не нужно создавать свои классы или обращаться к классам из библиотеки

Что такое классы обертки?  Классы которые представляют собой классы, помещающие примитивный тип в объект. Обертки для примитивных типов — это Double, Float, Long, Integer, Short, Byte, Character и Boolean.
позволяют в полной мере интегрировать примитивные типы в объектные типы ava.

Что такое автобоксинг и анбоксинг? Автоупаковка процесс автомитическиой упаковки примитивного типа в соответсвующий ему оболочку типа, каждый раз когда требуется объект этого типа. 
обратный процес называется авторсапаковка (анбоксинг)
