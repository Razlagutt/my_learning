пакет models :
	класс Item
		- имеет переменные экземпляра: id, имя клиента, описание, время создания, комментарии,
		- имеет методы: получения значения имени, описания, времени создания, комментария;
				 редактирования комментария;
	класс TaskInternet наследуется от Item
		- имеет свою переменную экземпляра
	класс TaskTelephony наследуется от Item
		- имеет переменную свою экземпляра
		
	класс Manager	
		- имеет переменную экземпляра: действие
		- имеет в себе методы  для работы с заявками 

пакет start
	класс Tracker
		содержит главный метод программы
		и меню вкотором будет реализован цикл для выбора действия
